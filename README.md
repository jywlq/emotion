# ☕ Cosmic Brew 星咖占卜

> 一杯咖啡的时间，探索内心的宇宙 ✨

Cosmic Brew 是一个**零配置、初学者友好的 Python 趣味项目**——走进一家虚拟的「宇宙咖啡馆」，让 AI 咖啡师为你解读性格、分析梦境、生成专属情绪星云。

---

## 🌟 四个"哇"时刻

| # | 体验 | 效果 |
|---|------|------|
| 1 | **专属星座图** | ASCII 星座 + 五维雷达图，一颗只有你才有的星星 |
| 2 | **情绪星云绽放** | 输入"开心"→金橙色星云；输入"忧郁"→蓝紫色星云 |
| 3 | **梦境坐标解读** | "正在连接宇宙信号..."仪式感动画 + 星际坐标 |
| 4 | **零配置开箱即用** | 不需要任何 API key，pip 安装两个依赖就能跑！ |

---

## 🚀 快速开始

```bash
# 1. 克隆/下载本项目
cd cosmic-brew

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行！
python3 cosmic_brew.py
```

**就这么简单！** 不需要注册、不需要 API key、不需要配置——直接运行就能体验完整功能。

## ☕ 使用指南

启动后，你会看到：

1. **🌌 欢迎动画** — 星星渐次亮起，宇宙咖啡馆招牌浮现
2. **☕ 点单** — 选择你的星空饮品（银河拿铁/星云摩卡/彗星美式/月光卡布）
3. **🔮 选择体验**：

   - **情绪星云**：输入今日心情 → 看到彩色 ASCII 星云 + 情绪建议
   - **梦境坐标**：描述梦境 → 获得仪式感解读 + 星际坐标

---

## 🔮 功能详解

### 1. 星座解读 (`🔮 星座解读`)

选择饮品 + 回答 3 个趣味问题，AI 咖啡师会根据你的选择生成：
- **五维性格分析**：温柔★☆☆☆☆ | 创造力★★★☆☆ | 直觉★★★★★ ...
- **专属星座图**：ASCII 艺术图案 + 星座名
- **AI 性格解读**：温暖诗意的个性化文字

> 💡 饮品的性格映射：银河拿铁→温柔型，星云摩卡→神秘型，彗星美式→探索型，月光卡布→直觉型

### 2. 情绪星云 (`🌙 情绪星云`)

输入今日心情（如"开心"、"有点焦虑"），AI 会：
- 识别情绪类别（开心/忧郁/焦虑/平静/愤怒）
- 用**彩色 ASCII 星云**可视化你的情绪（颜色随情绪变化！）
- 给出宇宙咖啡师的星象建议

### 3. 梦境坐标 (`🌌 梦境坐标`)

描述一个梦境，AI 会：
- 匹配梦境中的**象征符号**（飞翔→自由、水→情感、星星→希望...）
- 生成**仪式感等待动画**（"正在连接宇宙信号..."）
- 输出**星际坐标格式**的解读（如 `🌌 坐标: RA 14h 29m | Dec -62° 41'`）

---

## 🤖 双引擎 AI 设计

Cosmic Brew 采用**双引擎架构**，兼顾零门槛和深度体验：

| 引擎 | 何时使用 | 特点 |
|------|----------|------|
| **本地规则引擎**（默认）| 无 API key 时 | 100+ 解读模板，通过排列组合产生个性化输出 |
| **Gemini 引擎**（可选）| 设置 `GEMINI_API_KEY` 环境变量 | Google Gemini，更丰富的个性化解读 |

**启用 Gemini 增强（可选）：**

```bash
# 1. 安装 Gemini SDK
pip install google-generativeai

# 2. 获取免费 API Key：https://aistudio.google.com/apikey

# 3. 设置环境变量
export GEMINI_API_KEY="你的key"

# 4. 运行（会自动使用 Gemini 增强解读）
python3 cosmic_brew.py
```

## 📁 项目结构

```
cosmic-brew/
├── cosmic_brew.py          # 主入口 ⭐
├── test_cosmic.py          # 模块测试脚本
├── requirements.txt        # 依赖列表
├── README.md
├── ui/
│   ├── welcome.py          # 星空欢迎界面 + 动画
│   ├── menu.py             # 交互式菜单（questionary）
│   └── display.py         # 星座图/星云/解读渲染
├── core/
│   ├── personality.py      # 性格计算引擎
│   ├── dream.py            # 梦境解读引擎

│   └── templates.py        # 100+ 解读文本模板
├── ai/
│   ├── local_engine.py     # 本地规则引擎（默认）
│   └── gemini_engine.py    # Gemini API 引擎（可选）
└── data/
    ├── drinks.json         # 饮品数据 & 性格映射
    ├── questions.json      # 趣味问答库
    ├── dream_symbols.json  # 梦境象征词典
    └── star_patterns.json  # ASCII 星座图案库
```
